# v0.3 — Windows XP install USB

This file is archived. The old contents described the deleted legacy
three-tree FAT32 XP pipeline.

Current references:
- [`XP_FIRADISK_PIPELINE.md`](XP_FIRADISK_PIPELINE.md) — active
  GRUB4DOS + FiraDisk XP design.
- [`BACKLOG.md`](BACKLOG.md) — release blockers and follow-up work.
- [`HARDWARE_TESTS.md`](HARDWARE_TESTS.md) — hardware verification matrix.

Status as of 2026-05-21:
- `--type=windows-ntxp` is the active XP implementation.
- `--type=windows-xp` is a compatibility alias for `windows-ntxp`.
- There is no `windows-xp-legacy` CLI path.
- Windows 2000 media auto-detects as NT5-class, but Win2k install support
  is backlog work.
